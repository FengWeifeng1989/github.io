# 冯伟峰个人简历网站

本项目是基于冯伟峰个人简历内容创建的网页展示，托管在GitHub Pages上。

## 项目结构
- `index.html`: 主页
- `style.css`: 样式表
- `scripts.js`: JavaScript脚本
- 其他资源文件

## 功能特点
- 简历内容展示
- 数据预测模型展示（线性回归、多元回归等）
- 响应式设计
- 动态视觉效果
- 交互式数据模型展示
